# PRODIGY_FS_01

## Secure User Authentication System

### Description
A simple authentication system that allows users to:

- Register
- Login
- Access a protected dashboard
- Logout securely

### Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

### Features

- User Registration
- User Login
- Protected Route
- Session-like Authentication
- Logout Functionality

### How to Run

1. Open the project in VS Code.
2. Install Live Server.
3. Open `register.html` using Live Server.
4. Register a user.
5. Login and access the dashboard.
